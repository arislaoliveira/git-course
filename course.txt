git status - status do arquivo 

git add .\(nome do arquivo) - adicionar mudanças

git diff - o que foi alterado

git diff --staged (para arquivos que ja estao no staged)

(q para sair do git diff)

git commit -m "mensagem" - commitar mudanças

sync changes - envia para o repositório

git log - histórico dos commits

na parte da árvore (source control) é possível retornar a modificação 

git restore .\(nome do arquivo) - retorna a ultima modificação salva

git remote

git pull - merge com outros códigos

git branch testing - criar nova branch

git log --oneline --decorate - mostra commit e head

git checkout testing - ir para head testing
